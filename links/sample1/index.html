Introduction

Hyperlinks commonly follow the syntax

<a href="Link_url">Text</a>

For example:

<a href="https://www.google.com">Google</a>

is rendered like this: Google


Styling

By default a link is underlined

An unvisited link is blue

A visited link is purple

An active link is red


Attributes

The target attribute specifies where to open the link.

  • _blank opens in a new window or tab
  • _self opens in same window or tab
  • _parent opens in the parent frame
  • _top opens in the full window

Paths

The example above links to an external site, should you need to link to another page on your site, you use the file path. For example:

<a href="/html/about_me.html">About Me</a>

will link to the about_me.html file in the html folder

To link to another part of the same page, create an element with an id

<section id="projects">

and then you can create the link like this:

<a href="#projects>
